Bengali Fruits

Bengali Fruits is an intuitive app that will help the user to learn the various fruits name in two languages, Bengali and English. Borno Quizz

BornoQuizz is the application which will help your kids to learn the Bangla Bornomala (Bengali Letters) in an intuitive way. It has well designed graphical contents that will engage the kids for better learning. It has two modes, learning mode and quiz mode. Learning mode will teaches the bornomala with proper pronunciation and quiz mode will test your kids know-how of those bornomala. It teaches both Swarborno and Benjonborno the two form of letters used to write in Bengali. Kids will have fun during the learning of Bangla Bornomalas.

  • Learning Mode
